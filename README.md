# ERP – Software Architecture Workshop

Este repositorio contiene el desarrollo del taller de Arquitectura de Software
para un sistema ERP, utilizando metodología ágil, diagramas C4 y la plantilla arc42.

## Contenido
- Gestión ágil del proyecto en Jira
- Documentación de arquitectura con arc42
- Diagramas UML y C4 realizados con PlantUML
